# Quick Start

## 1. 本地开发启动

```bash
npm install
npm run dev
```

打开：

```text
http://localhost:3009
```

## 2. Windows 一键启动

直接运行：

- `start.bat`
- `start.ps1`

## 3. Docker 一键启动

Mac 或有 Docker 环境的机器可直接运行：

- `docker-start.command`

更新：

- `docker-update.command`

删除容器：

- `docker-remove.command`

彻底删除容器和本地数据库：

- `docker-remove-all.command`

## 4. 主要页面

- `/`：客户演示主站
- `/console`：交互控制台

## 5. Demo 口径

- 平台：`Mendix`
- 文档解析：`Docling`
- LLM：`DeepSeek`
- Embedding：`BAAI/bge-m3`
- 知识库：`PostgreSQL + pgvector`
- 闭环：`Mendix Workflow`
